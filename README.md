# ASCII Art Maker

Yazdığınız kelimeyi ASCII Art şekilde size geri gönderen, Python ile yapılan bir kod.

**Gereksinimler:** **Windows ve Mac için:** pip install art <br />
                   **Linux için:** python3 -m venv myvenv <br />
                                  source myvenv/bin/activate <br />
                                  pip3 install art
